• Hjem

Pitch - Jørgen Hei, jeg heter Jørgen. For tiden er jeg kursdeltaker hos Jobloop i kurset Kodehode. I kurset har jeg fått en innføring i HTML/CSS samt programmeringsspråket JavaScript og JavaScript rammeverkene ReactJS og backend teknologien NodeJS.

Tidligere har jeg jobbet som praktikant i butikkjedene Clas Ohlson, Spaceworld og Meny. Samt praktikant som IT-driftsmedarbeider og tech support.

Som IT-driftsmedarbeider jobbet jeg med forskjellige server-teknologier: Windows Server 2007 - 2019, FileZilla FTP, Windows Server Datacenter og Windows Active Directory. Samt tech support i form av PC-reparasjoner, Mobiltelefon oppsett for bedrift og ellers tekniske små oppgaver som printer feil, feilsøking i operativsystem og skriving av brukermanualer, fillagring og server administrasjon.

Mine interesser og hobbyer er å spille gitar og keyboard, drive med programmering, spille videospill og å lage musikk på amatørnivå med programvaren Ableton Live.

Jeg begynte å spille gitar som 12-åring, og har drevet jevnt med å øve på dette i mange mange år, og har fremdeles mye glede av instrumentet. Å lære forskjellige instrumenter og programvare med spesifikke funksjoner, har lært meg å lære på egenhånd.
